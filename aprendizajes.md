¿Que es git y para que sirve?
Es un sistema de control de versiones diseñado para rastrear cambios
en cualquier conjuntos de archivos y se utiliza para coordinar el trabajo entre desarrolladores. 
Sirve para revertir errores, trabajo en equipos y tener un historiales detallados.

Los 5 comandos mas usados seran:
1) git status
2) git commit
3) git add
4) git push
5) git checkout
